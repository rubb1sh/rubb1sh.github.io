git命令大全

发表于 2019-08-21   |   分类于 git

赶紧常用的git命令需要背一下了，不然每次想起命令不知道具体是什么。所以找了一个看起来还行的博客，把命令背一下。add comment这种太常用的就不写了，写一些不是太常用的。

原文链接

  1. git branch: 查看分支。-a查看所有分支，-r查看所有本地分支。
  2. git remote show origin: 查看远程资源
  3. git checkout: 切换分支。-b创建新的分支，并切换到这个分支。-d删除分支。
  4. git merge [name]: 将名为name到分支与当前分支合并
  5. git log: 查看commit日志
  6. git diff: 查看尚未暂存的更新（没有add的改变）
  7. git stash push: 把当前未提交的修改保存在一个临时空间。
  8. git stash pop: 把保存在临时空间的修改抓去下来（恢复）。git stash的应用场景主要是针对当你需要回退版本，但是当前暂存区的内容又需要保存下来时使用。这样可以在你回退版本后，在返回当前。这样就避免了，你为了回退版本，而进行“脏提交”。
  9. git stash apply: 把多个缓存堆栈应用到工作目录，并且不会删除stash拷贝。（pop会删除拷贝，因为他是pop）。
  10. git stash后面可以附加一些命令，因此可以把git stash看作一个基础操作进行拓展。后面加list，是列出所有缓存堆栈数据。drop是移除。show查看查看和工作区的diff。
