​ 在macOS下安装mysql数据库有两种方式，1种是通过使用brew安装，1种是通过去mysql官网下载包安装。

使用后者安装的好处是，mysql的安装包会自动配置好大部分内容，并且能在系统偏好设置中管理mysql数据库的开关。
